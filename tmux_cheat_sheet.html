TMUX CHEAT SHEET

Session · Window · Pane · Copy Mode 快速參考

預設前綴鍵 Prefix = Ctrl-b
Session 管理
Prefix $重新命名當前 session
Prefix dDetach（背景執行，離開不中斷）
Prefix s列出並切換 session（互動選單）
Prefix (切換至上一個 session
Prefix )切換至下一個 session
Prefix L切換至上一個使用的 session
Prefix $重新命名 session
Window（標籤）管理
Prefix c建立新視窗
Prefix ,重新命名當前視窗
Prefix &關閉當前視窗（需確認）
Prefix n切換至下一個視窗
Prefix p切換至上一個視窗
Prefix l切換至上一個使用的視窗
Prefix 0-9切換至指定編號視窗
Prefix w列出所有視窗（互動選單）
Prefix f依名稱搜尋視窗
Prefix .移動視窗至指定編號
Pane 分割與切換
Prefix %垂直分割（左右）
Prefix "水平分割（上下）
Prefix x關閉當前 pane（需確認）
Prefix !將當前 pane 移為獨立視窗
Prefix ↑↓←→切換至相鄰 pane
Prefix o循環切換 pane
Prefix ;切換至上一個使用的 pane
Prefix q顯示 pane 編號（按編號跳躍）
Prefix z最大化 / 還原當前 pane（zoom）
Prefix {將當前 pane 向前移動
Prefix }將當前 pane 向後移動
Prefix Space循環切換 pane 排列方式
Prefix Alt-↑↓←→調整 pane 大小（5 格）
Prefix Ctrl-↑↓←→調整 pane 大小（1 格）
Pane 同步 / 佈局
Prefix Alt-1佈局：even-horizontal（水平均分）
Prefix Alt-2佈局：even-vertical（垂直均分）
Prefix Alt-3佈局：main-horizontal
Prefix Alt-4佈局：main-vertical
Prefix Alt-5佈局：tiled（平鋪）
:setw synchronize-panes on同步所有 pane 輸入
:setw synchronize-panes off關閉同步輸入
Copy Mode（vi 模式）
Prefix [進入 Copy Mode
Prefix ]貼上剪貼簿內容
Prefix PgUp進入 Copy Mode 並向上捲動
h j k l移動游標
w / b跳至下 / 上一個單字
/{pattern}向下搜尋
?{pattern}向上搜尋
n / N下 / 上一個搜尋結果
g / G跳至頂端 / 底端
Ctrl-u / d向上 / 下捲動半頁
Ctrl-b / f向上 / 下捲動一頁
Space開始選取
Enter複製選取並離開 Copy Mode
v開始字元選取（需設定 vi 模式）
V開始行選取
Ctrl-v開始區塊選取
y複製選取（yank）
q / Esc離開 Copy Mode
其他快捷鍵
Prefix ?列出所有快捷鍵
Prefix :進入 tmux command prompt
Prefix t在當前 pane 顯示時鐘
Prefix i顯示當前視窗資訊
Prefix ~顯示 tmux 訊息記錄
Prefix r重新載入設定（需設定於 .tmux.conf）
Prefix D選擇要 detach 的 client
Prefix Ctrl-z暫停 tmux client
:source ~/.tmux.conf重新載入設定檔
:show-options -g顯示全域設定
:set -g mouse on啟用滑鼠支援
Shell CLI 指令
tmux啟動新 session
tmux new -s {name}建立並命名新 session
tmux attach重新連接最近的 session
tmux attach -t {name}連接指定 session
tmux ls列出所有 session
tmux kill-session -t {name}刪除指定 session
tmux kill-server終止 tmux server 及所有 session
tmux new-window建立新視窗（CLI）
tmux split-window -h垂直分割 pane
tmux split-window -v水平分割 pane
tmux send-keys -t {target} "{cmd}" Enter向指定 pane 發送指令
tmux rename-session -t {old} {new}重新命名 session
tmux swap-pane -s {src} -t {dst}交換兩個 pane
tmux move-window -t {session}移動視窗至另一個 session
tmux show-keys顯示所有快捷鍵綁定
tmux info顯示所有 session / client 資訊
~/.tmux.conf 常用設定
set -g prefix C-a將前綴鍵改為 Ctrl-a
set -g mouse on啟用滑鼠（點擊切換 pane / 捲動）
set -g history-limit 50000增加捲動歷史行數
set -g base-index 1視窗從 1 開始編號
setw -g pane-base-index 1Pane 從 1 開始編號
setw -g mode-keys viCopy Mode 使用 vi 按鍵
set -g default-terminal "tmux-256color"啟用 256 色支援
set -g renumber-windows on關閉視窗後自動重新編號
set -g status-position top將狀態列移至頂端
set -sg escape-time 0消除 Esc 延遲（Neovim 適用）
沒有符合的指令
