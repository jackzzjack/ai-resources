主流 AI 模組家族

目前市場上由科技巨頭與新創公司主導的幾大模組系列，各有其擅長的領域。

OpenAI (GPT 系列)
最具代表性的模型，綜合能力最強，生態系最完整。
GPT-4o o1 (推理模型) 多模態 最強 API 支持
Anthropic (Claude 系列)
以文字風格自然、邏輯細膩著稱，程式碼能力極強，深受開發者喜愛。
Claude 3.5 Sonnet Claude 3.5 Haiku 超長 Context 視覺理解
Google (Gemini 系列)
擁有最長的上下文窗口（Context Window），深度整合 Google 生態系。
Gemini 1.5 Pro Gemini 1.5 Flash 2M 上下文 原生多模態
Meta (Llama 系列)
開源模組的領導者，可用於本地部署，性能接近閉源頂尖模組。
Llama 3.1 Llama 3.2 (視覺) 開源 / 本地部署 社群支持廣
DeepSeek (深度求索)

OpenRouter：模型聚合平台

OpenRouter 是一個統一的 API 閘道，讓你可以透過單一介面存取幾乎所有主流的 AI 模型。

為什麼使用 OpenRouter？

傳統上，如果你想同時使用 GPT-4、Claude 3.5 和 Llama 3，你需要分別向 OpenAI、Anthropic 和 Meta (或雲端供應商) 申請 API Key。OpenRouter 解決了這個痛點。

統一 API
使用單一 API Key 即可調用超過 100 種模型，無需維護多個帳戶。
價格透明
OpenRouter 通常直接傳遞供應商的價格，有時甚至更便宜（透過大量採購）。
高可用性
當某個模型供應商（如 Anthropic）當機時，OpenRouter 可自動切換到其他供應商提供的相同模型。
功能 官方 API OpenRouter
Key 管理 每個廠商一個 Key 一個 Key 搞定全部
支付方式 需綁定多張信用卡 統一儲值（支援加密貨幣）
模型覆蓋 僅限該廠商 涵蓋 OpenAI, Anthropic, Google, Meta, DeepSeek 等
延遲 最低 (直連) 極低 (多一層轉發，體感不明顯)

我該選哪一個模型？

根據你的具體使用情境，我們給出以下建議：

日常全能助理
首選 GPT-4o。回答速度快，多模態能力強，且 ChatGPT App 體驗極佳。
程式開發 / 重度寫作
首選 Claude 3.5 Sonnet。它是目前公認最強的 coding 助手，語氣也最像人類。
超長文件 / 論文分析
首選 Gemini 1.5 Pro。你可以把一整本書或一小時的影片丟給它，它能精準回答細節。
開發預算有限
首選 DeepSeek-V3。能力接近 GPT-4o，但 API 價格僅為其幾十分之一。
