MINICOM CHEAT SHEET

Serial Terminal 快速參考 · 嵌入式開發 / UART 通訊

預設 Escape = Ctrl-A（按下後再按指令鍵）
基本導覽（Ctrl-A 前綴）
Ctrl-A Z開啟說明選單（Help menu）
Ctrl-A X離開 minicom（Exit）
Ctrl-A Q離開但不重設 modem（Quit）
Ctrl-A O設定選單（Configuration）
Ctrl-A M傳送 modem 初始化字串
Ctrl-A R接收檔案（Receive）
Ctrl-A S傳送檔案（Send）
Ctrl-A I切換 Line Wrap 模式
Ctrl-A U切換新行模式（Add LF）
Ctrl-A E切換本地回顯（Local Echo）
Ctrl-A C清除螢幕
Ctrl-A B傳送 BREAK 訊號
Ctrl-A F傳送換頁字元（FF）
Ctrl-A G執行 script 檔案
Ctrl-A K執行 Kermit 協定
Ctrl-A H手動撥號（Hang up）
Ctrl-A D撥號目錄（Dial directory）
Ctrl-A A傳送 Ctrl-A 字元本身
捲動 / 緩衝區
Ctrl-A J進入捲動緩衝模式（Scroll Back）
↑ / ↓（捲動模式）上 / 下一行
PgUp / PgDn（捲動模式）上 / 下一頁
/ {pattern}（捲動模式）搜尋字串
n（捲動模式）下一個搜尋結果
q / Esc離開捲動模式
Serial 連線設定
Ctrl-A O進入設定選單
Serial port setup設定序列埠（A 改裝置路徑）
A設定裝置路徑（如 /dev/ttyUSB0）
B設定 Baud Rate（如 115200）
C設定資料位元（Data bits，常用 8）
D設定停止位元（Stop bits，常用 1）
E設定同位元（Parity，常用 None）
F設定硬體流控（Hardware FC）
G設定軟體流控（Software FC，XON/XOFF）
Save setup as dfl儲存為預設設定
Save setup as...儲存為指定名稱設定檔
Exit離開設定並套用
常用 Baud Rate / 設定組合
9600常用於舊設備、BIOS console
38400部分嵌入式裝置
57600Arduino 部分設定
115200最常見預設（Linux kernel、Arduino）
230400高速嵌入式應用
1000000+高速 UART（視硬體支援而定）
8N18 data bits, No parity, 1 stop bit（最常用）
8E18 data bits, Even parity, 1 stop bit
7N17 data bits, No parity, 1 stop bit
檔案傳輸協定
Ctrl-A S傳送檔案（選擇協定）
Ctrl-A R接收檔案（選擇協定）
Xmodem簡單傳輸，128 Byte 區塊，相容性高
Ymodem支援批次傳輸與 1K 區塊
Zmodem最常用，支援斷點續傳、較快速
Kermit跨平台傳輸協定，適用特殊環境
ASCII純文字傳輸（無錯誤校正）
sz {file}遠端：以 Zmodem 傳送檔案至本機
rz遠端：以 Zmodem 接收本機上傳的檔案
擷取 / 記錄
Ctrl-A L開啟 / 關閉 Log 擷取至檔案
Ctrl-A P傳送目前緩衝區至印表機 / 檔案
Capture on開始記錄所有收到的資料
Capture off停止記錄
Capture pause暫停記錄
-C {file}啟動時直接開始擷取至指定檔案
CLI 啟動參數
minicom以預設設定啟動
minicom -s啟動並直接進入設定選單
minicom -D /dev/ttyUSB0指定裝置路徑
minicom -b 115200指定 Baud Rate
minicom -8使用 8 位元字元（UTF-8 支援）
minicom -o不初始化 modem（直接連線）
minicom -c on啟用彩色顯示
minicom -C {file}啟動並擷取至指定 log 檔
minicom -r倒轉（翻轉）BS 與 DEL 鍵行為
minicom {config}載入指定名稱的設定（~/.minirc.{config}）
minicom -S {script}啟動並執行指定 script
ls /dev/ttyUSB*列出 USB-Serial 裝置
ls /dev/ttyACM*列出 ACM 裝置（Arduino 等）
ls /dev/ttyS*列出實體 COM Port
dmesg | grep tty從 dmesg 確認裝置名稱
裝置權限設定
ls -l /dev/ttyUSB0查看裝置所屬群組（通常為 dialout）
sudo usermod -aG dialout $USER將使用者加入 dialout 群組
sudo usermod -aG uucp $USERArch Linux 系列使用 uucp 群組
sudo chmod 666 /dev/ttyUSB0臨時開放裝置讀寫（不建議長期）
groups確認當前使用者所屬群組（需重新登入後生效）
使用技巧 / 常見問題
Hardware FC 關閉大多數 USB-Serial 不需硬體流控，出現亂碼時先關閉
Local Echo 開啟輸入看不到字元時，以 Ctrl-A E 開啟回顯
Add LF 開啟換行只顯示一行時，以 Ctrl-A U 加入 LF
Line Wrap 開啟長行被截斷時，以 Ctrl-A I 切換換行
minirc 設定位置預設設定存於 ~/.minirc.dfl
/etc/minicom/系統全域設定目錄
Ctrl-A A若需傳送真正的 Ctrl-A，使用此組合
Ctrl-]部分設定中用於替代前綴鍵
picocom 替代方案picocom -b 115200 /dev/ttyUSB0（更輕量）
screen 替代方案screen /dev/ttyUSB0 115200
cu 替代方案cu -l /dev/ttyUSB0 -s 115200
沒有符合的指令
