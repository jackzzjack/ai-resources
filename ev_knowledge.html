整體架構 — EV 系統概覽

電動車由高壓動力系統、低壓輔助系統、熱管理系統與機械傳動系統共同組成。各子系統之間透過 CAN bus / Ethernet 進行通訊協調。

🔋 高壓電池包 HV Battery Pack
400V / 800V
由數百至數千顆電芯串並聯組成，電壓通常為 400V 或 800V 系統。包含 BMS 進行狀態監控與保護。
⟶
🔄 逆變器 Inverter
DC → AC
將電池輸出的直流電（DC）轉換為三相交流電（AC），驅動電動馬達。同時在回生制動時將 AC 轉回 DC 給電池充電。
⟺
⚙️ 電動馬達 Electric Motor
PMSM / IM
將電能轉換為機械旋轉力。常見類型：永磁同步馬達（PMSM）效率高；感應馬達（IM）耐用性佳。回生制動時反向運作為發電機。
⟶
🔩 減速齒輪箱 Gearbox
固定單速
EV 通常使用固定比例單速減速箱（約 7:1 至 12:1），無需換檔。將馬達高轉速轉換為輪軸適合的低轉速與高扭矩。
⟶
🚗 車輪 Wheels
最終驅動輸出。雙馬達 AWD 車型在前後軸各配置一套逆變器 + 馬達，可獨立控制扭矩分配，實現扭矩向量控制。
↕ 回生制動時能量反向流動（Wheels → Motor(Generator) → Inverter → Battery）
輔助子系統
🔌 車載充電器 OBC
AC → DC
Onboard Charger：將家用或公共 AC 電源轉換為 DC 對高壓電池充電。功率通常 3.3kW ~ 22kW。DC 快充時繞過 OBC，直接輸入電池。
⬇️ DC-DC 轉換器 DC-DC Conv.
400V → 12V
將高壓電池電壓步降至 12V，供應車上所有低壓電子設備（燈光、音響、ECU、感測器等），取代傳統燃油車的交流發電機角色。
🔋 12V 輔助電池 12V Aux Battery
鉛酸 / LFP
提供車輛靜止時的低壓電力維持（門鎖、防盜、接收遠端指令），以及高壓系統啟動前的初始電力。現代 EV 逐漸採用 LFP 小電池取代傳統鉛酸。
🌡️ 熱管理系統 TMS
電池 / 馬達 / 駕駛艙
控制電池、馬達、逆變器的工作溫度。包含液冷迴路、電動壓縮機、PTC 加熱器或熱泵。電池最佳工作溫度約 20~35°C。
📡 VCU / BMS 整車控制器
電池管理系統
VCU（Vehicle Control Unit）統籌協調所有子系統。BMS 即時監控每顆電芯的電壓、溫度、SOC、SOH，執行均衡與保護策略。
⚡
高壓系統 (HV)
電池包輸出電壓通常為 400V（標準）或 800V（高性能/快充）。高壓系統包含電池、逆變器、馬達、OBC、電動壓縮機、PTC 加熱器。高壓線束採用橙色標識，需嚴格安全隔離。
400V 系統 800V 系統 橙色線束
🔵
低壓系統 (LV 12V)
由 DC-DC 轉換器從高壓側降壓供電。覆蓋範圍：車身控制模組（BCM）、燈光、雨刷、車窗升降、中控娛樂、ADAS 感測器、ECU 網路（CAN/LIN/Ethernet）。
12V / 48V CAN Bus BCM
❄️
熱管理系統 (TMS)
整合電池冷卻、馬達冷卻、座艙空調三個液冷迴路。先進熱泵系統可在低溫下從環境取熱，大幅減少冬季電耗。電池預熱功能改善低溫充電接受度。
熱泵 液冷 電動壓縮機
🧠
整車電子架構
現代 EV 採用集中式 E/E 架構，以 1~3 顆中央計算域控制器（Domain ECU / Zonal ECU）取代傳統數十顆分散 ECU。車用乙太網路（100BASE-T1 / 1000BASE-T1）傳輸大頻寬數據。

電池包構造 — Cell → Module → Pack

電動車電池系統採用三層式架構，從最小的電芯單元逐步組合為模組，再整合成電池包。BMS 全程監控，確保安全與效能。

動力傳遞 — 逆變器、馬達與扭矩

電能從電池經逆變器轉換後驅動電動馬達，產生扭矩推動車輛。制動時馬達反向運作回收動能。

動力流程圖
🔋 DC 電池 ~400V 直流
高壓電池包輸出直流電（DC），電壓視 SOC 而浮動，400V 系統約 320~420V，800V 系統約 650~820V。
⟶
🔄 逆變器 DC → 三相 AC
IGBT / SiC MOSFET
使用 PWM（脈寬調變）技術，透過高速切換功率半導體（IGBT 或 SiC MOSFET）將 DC 轉換為可調頻調壓的三相 AC。SiC 切換頻率更高、損耗更低。
⟺
⚙️ 三相 AC 馬達 電能 → 機械能
PMSM / IM
接受逆變器輸出的三相交流電，透過電磁感應產生旋轉力（扭矩）。制動時反向運作，旋轉動能轉為電能（回生制動）。
⟶
⚙️ 減速箱 固定齒比
約 8:1~10:1
EV 無需多檔變速箱。固定減速比將馬達高轉速（最高 15000~20000 RPM）降至車輪適用轉速，同時放大扭矩。
⟶
🚗 驅動軸/車輪 機械輸出
透過半軸（CV joint）將扭矩傳遞至驅動輪。電子差速功能在轉彎時自動調節左右輪扭矩分配，無需機械差速器。
回生制動路徑（反向）：車輪旋轉 → 馬達（發電機模式）→ 逆變器（整流 AC→DC）→ 高壓電池充電。可回收 10~30% 制動能量，配合液壓制動由 ABS/ESC 系統協調混合制動。
馬達類型比較

⚡ 永磁同步馬達 PMSM

轉子嵌有永久磁鐵（釹鐵硼），與定子旋轉磁場同步轉動。
優點
• 效率極高（90~98%）
• 功率密度高，體積小
• 低轉速大扭矩響應快
缺點
• 依賴稀土材料（Nd 釹）
• 高速弱磁控制複雜
應用：Tesla Model 3/Y 後馬達、BYD、Hyundai Ioniq

🔄 感應馬達 Induction Motor (IM)

依靠定子旋轉磁場在轉子導體產生感應電流，形成驅動力矩（非同步）。
優點
• 結構簡單，成本低
• 無稀土依賴
• 高速性能佳，耐用
缺點
• 效率略低於 PMSM
• 需要電流維持磁場（功率因數低）
應用：Tesla Model S/X 前馬達（雙馬達 AWD 搭配）
EV 扭矩特性曲線（示意圖）
EV 扭矩特性：從零轉速即可輸出最大扭矩（恆扭矩區），提供優異的起步加速性能。超過額定轉速後進入「弱磁」控制，功率維持恆定，扭矩隨轉速提升而下降。相較於內燃機需達特定轉速才有峰值扭矩，EV 的扭矩響應更直接。

保護電路概念
🛡️
主繼電器 Main Relay
高壓電池正負極各一顆，碰撞感測或異常時立即斷開高壓回路。預充繼電器（Pre-charge Relay）與限流電阻串聯，防止電容器組瞬間大電流衝擊。
💥
手動維修開關 MSD
Manual Service Disconnect：維修時由技師手動拔除，物理隔離高壓電路中段，消除觸電風險。通常位於電池包中央便於存取位置。
⚡
絕緣監測 IMD
Insulation Monitoring Device：持續監測高壓系統對車身金屬底盤的絕緣電阻（應 >500Ω/V）。絕緣降低時立即告警並可觸發保護動作，防止觸電事故。
🔧
高壓互鎖 HVIL
High Voltage Interlock Loop：貫穿所有高壓連接器的低壓監測信號線路。任何連接器鬆脫時信號中斷，BMS/VCU 立即切斷高壓電源，防止帶電插拔意外。
高壓安全顏色標準（ISO 6469）：所有 HV 線束使用橙色外皮，連接器使用橙色標識。維修人員必須使用 1000V 等級絕緣工具，並確認 HV 系統完全放電（電容放電時間約 5 分鐘）後才可進行維修作業。

充電系統 — AC 慢充、DC 快充與 V2G

EV 充電系統涵蓋從家用 Level 1 到超高速 DC 快充的完整功率範圍，以及未來的雙向 V2G 能源交互技術。

AC 交流充電 (Level 1 / Level 2)
🏠 電網 AC 110V/220V/380V
家用/公共
Level 1：110~120V 單相（北美）。Level 2：220~240V 單相或三相 AC。
⟶
🔌 充電槍/EVSE 充電站/充電盒
CP 信號通訊
EVSE（Electric Vehicle Supply Equipment）提供 AC 電源並透過 Pilot 信號（CP）與車輛握手，確認額定電流後才開始充電。
⟶
⚡ OBC 車載充電器
AC → DC 整流
Onboard Charger 將 AC 整流、功因校正（PFC）後輸出穩定 DC 電壓，支援 3.3kW ~ 22kW 功率範圍。功率越大體積越重。
⟶
🔋 高壓電池 CC-CV 充電
BMS 監控
BMS 協調充電電流曲線：先恆流（CC）快速充入，近滿電時切換恆壓（CV）緩充，保護電芯延長壽命。通常充至 80% 較快，80~100% 速度大幅降低。
DC 直流快充（繞過 OBC）
⚡ 電網 AC 三相高壓
380V / 10kV
DC 快充站需要三相高壓供電，功率 50kW~350kW 等級需工業電力基礎設施支援。
⟶
🏭 站端充電模組 Off-board Charger
整流 + DCDC
快充站內建大功率整流器與 DC-DC 轉換器，將 AC 轉為可調電壓的 DC 直接輸出。模組化設計支援功率動態分配。
⟶
🔌 DC 充電槍 CCS / CHAdeMO
NACS / GB/T
DC 快充直接輸出高壓 DC（最高 1000V），繞過車上 OBC，直接進入電池組。通訊協議：CCS 使用 PLC、CHAdeMO 使用 CAN。
⟶
🔋 高壓電池 直接入電
BMS 控制電流
BMS 向充電站即時傳送充電需求（電壓、最大電流）。電芯溫度、電壓均衡狀態影響最大接受電流，BMS 動態調節保護電芯。

🏠 家庭/電網 V2H / V2G

V2G 效益：削峰填谷穩定電網 | 車主獲得電力補償收益 | 提升再生能源利用率
挑戰：需雙向 OBC（成本較高）| 電網通訊標準統一 | 頻繁充放影響電池壽命（需 BMS 最佳化）
