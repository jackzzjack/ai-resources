游標移動
h j k l左 / 下 / 上 / 右

b跳至上一個單字開頭
B跳至上一個 WORD 開頭
e跳至當前單字結尾
ge跳至上一個單字結尾
0行首（column 0）
^行首第一個非空白字元
$行尾
g_行尾最後非空白字元
gg跳至檔案開頭
G跳至檔案結尾
{n}G跳至第 n 行
:{n}跳至第 n 行（command line）
Ctrl-d向下捲動半頁
Ctrl-u向上捲動半頁
Ctrl-f向下捲動一頁
Ctrl-b向上捲動一頁
zz將游標行置於螢幕中央
zt將游標行置於螢幕頂端
zb將游標行置於螢幕底端
%跳至對應的括號 / 標籤
{跳至上一段落
}跳至下一段落
H跳至螢幕最上方
M跳至螢幕中間
L跳至螢幕最下方
字元跳躍 / 文字物件
f{c}跳至行內下一個字元 c
F{c}跳至行內上一個字元 c
t{c}跳至字元 c 之前
T{c}跳至字元 c 之後（反向）
;重複上一個 f/F/t/T
,反向重複 f/F/t/T
iw內部單字（inner word）
aw整個單字（含空白）
is內部句子
ip內部段落
i"雙引號內（inner "…"）
a"含雙引號整體
i(括號內（inner (…)）
a(含括號整體
i{大括號內
itHTML/XML 標籤內
進入插入模式
i在游標前插入
I在行首第一個非空白字元前插入
a在游標後附加
A在行尾附加
o在下方新增一行並進入插入
O在上方新增一行並進入插入
s刪除游標字元並插入
S刪除整行並插入
C刪除至行尾並插入
gi回到上次插入位置
Esc返回 Normal 模式
Ctrl-[返回 Normal 模式（等同 Esc）
Ctrl-c返回 Normal 模式（不觸發縮寫）
Ctrl-w刪除上一個單字
Ctrl-u刪除至行首
Ctrl-t增加縮排
Ctrl-d減少縮排
Ctrl-n自動補全（下一項）
Ctrl-p自動補全（上一項）
刪除 / 修改
x刪除游標下字元
X刪除游標前字元
dd刪除整行
dw刪除至下一個單字
de刪除至單字結尾
d$刪除至行尾
d0刪除至行首
diw刪除內部單字
daw刪除含空白的整個單字
dip刪除整個段落
{n}dd刪除 n 行
r{c}替換單一字元為 c
R進入 Replace 模式（覆蓋輸入）
cw修改至單字結尾並進入插入
cc修改整行
c$修改至行尾
ciw修改內部單字
ci"修改雙引號內容
ci(修改括號內容
~切換大小寫
gu{m}將動作 m 範圍轉為小寫
gU{m}將動作 m 範圍轉為大寫
g~{m}切換動作 m 範圍大小寫
複製 / 貼上 / 撤銷
yy複製整行（yank）
yw複製至下一個單字
y$複製至行尾
yiw複製內部單字
yap複製整個段落
{n}yy複製 n 行
p在游標後貼上
P在游標前貼上
gp貼上後游標移至貼上內容末尾
"]p貼上並調整縮排
u撤銷（undo）
Ctrl-r重做（redo）
.重複上一次修改
>>向右縮排
<<向左縮排
==自動縮排當前行
={m}自動縮排動作 m 範圍
gg=G自動縮排整個檔案
J合併下一行至當前行
gJ合併下一行（不加空格）
Visual 模式
v進入字元視覺模式
V進入行視覺模式
Ctrl-v進入區塊視覺模式
gv重新選取上次選取範圍
o切換選取範圍的錨點端
d刪除選取
y複製選取
c修改選取（刪除並插入）
r{c}將選取全部替換為字元 c
~切換選取大小寫
u選取轉小寫
U選取轉大寫
>縮排選取
<反縮排選取
!將選取傳入外部指令過濾
I區塊模式：在每行前插入（Ctrl-v 後）
A區塊模式：在每行後附加
搜尋
/{pattern}向下搜尋 pattern
?{pattern}向上搜尋 pattern
n移至下一個搜尋結果
N移至上一個搜尋結果
*搜尋游標下的整個單字（向下）
#搜尋游標下的整個單字（向上）
g*搜尋游標下的單字（部分符合）
gd跳至局部變數定義處
gD跳至全域變數定義處
:noh清除搜尋高亮
:set hlsearch啟用搜尋高亮
:set ignorecase搜尋忽略大小寫
:set smartcase輸入大寫時區分大小寫
替換（Substitute）
:s/old/new/替換當前行第一個符合
:s/old/new/g替換當前行所有符合
:%s/old/new/g替換整個檔案所有符合
:%s/old/new/gc替換前逐一確認
:%s/old/new/gi忽略大小寫替換
:5,12s/old/new/g替換第 5–12 行
:'<,'>s/old/new/g替換 Visual 選取範圍
&重複上一次 :s 替換（當前行）
g&重複上一次 :s 替換（整個檔案）
檔案 / 緩衝區
:w儲存檔案
:w {file}另存新檔
:wq儲存並離開
:wqa儲存所有緩衝區並離開
:q離開（若有未儲存修改則失敗）
:q!強制離開（捨棄修改）
:qa!強制關閉所有視窗
ZZ儲存並離開（同 :wq）
ZQ不儲存離開（同 :q!）
:e {file}開啟檔案
:e!重新載入（捨棄修改）
:ls列出所有緩衝區
:b{n}切換至第 n 個緩衝區
:bn切換至下一個緩衝區
:bp切換至上一個緩衝區
:bd刪除（關閉）當前緩衝區
:r {file}將檔案內容插入游標後
:r !{cmd}將指令輸出插入游標後
分割視窗 / 分頁
:sp水平分割視窗
:vsp垂直分割視窗
Ctrl-w s水平分割（同 :sp）
Ctrl-w v垂直分割（同 :vsp）
Ctrl-w h/j/k/l切換至左/下/上/右視窗
Ctrl-w w循環切換視窗
Ctrl-w H/J/K/L移動視窗至邊緣
Ctrl-w =均等視窗大小
Ctrl-w +/-增加 / 減少視窗高度
Ctrl-w >/<增加 / 減少視窗寬度
Ctrl-w q關閉當前視窗
Ctrl-w o只保留當前視窗
:tabnew新增分頁
:tabn下一個分頁
:tabp上一個分頁
:tabc關閉當前分頁
gt切換至下一個分頁
gT切換至上一個分頁
巨集 / 重複
q{r}開始錄製巨集至暫存器 r
q停止錄製巨集
@{r}執行暫存器 r 的巨集
@@重複上一次巨集
{n}@{r}執行巨集 n 次
.重複上一次 Normal 模式修改
@:重複上一次 Ex 指令
:g/{p}/d刪除所有符合 pattern 的行
:g/{p}/{cmd}對所有符合行執行 cmd
:v/{p}/{cmd}對所有不符合行執行 cmd
標記（Marks）
m{a-z}設置局部標記（小寫字母）
m{A-Z}設置全域標記（大寫字母，跨檔案）
`{a}跳至標記 a 的精確位置
'{a}跳至標記 a 所在行的行首
``跳至上一個跳躍位置
`.跳至上一次修改位置
`^跳至上一次插入結束位置
`[跳至上一次修改或複製的起點
`]跳至上一次修改或複製的終點
`<跳至上次 Visual 選取起點
`>跳至上次 Visual 選取終點
:marks列出所有標記
:delmarks {a}刪除標記 a
:delmarks!刪除所有小寫標記
Ctrl-o跳至跳躍列表上一個位置
Ctrl-i跳至跳躍列表下一個位置
:jumps顯示跳躍列表
暫存器（Registers）
"{r}y複製至暫存器 r
"{r}p從暫存器 r 貼上
"{r}d刪除至暫存器 r
"無名暫存器（上次操作）
"0複製暫存器（yank register）
"1–"9刪除歷史暫存器（1 最新）
"a–"z命名暫存器（可長期保存）
"A–"Z追加至對應小寫命名暫存器
"+系統剪貼簿（clipboard）
"*主選取區（primary selection）
"_黑洞暫存器（丟棄）
"/搜尋暫存器（上一次搜尋）
":上一次 Ex 指令
"%當前檔案名稱
".上一次插入的文字
:reg列出所有暫存器內容
Ctrl-r {r}（插入模式）插入暫存器 r 的內容
外部指令 / 實用
:!{cmd}執行外部 shell 指令
:shell暫時切換至 shell
Ctrl-z暫停 Vim（fg 回到 Vim）
:make執行 make 並捕捉錯誤
:cn跳至下一個 quickfix 錯誤
:cp跳至上一個 quickfix 錯誤
:copen開啟 quickfix 視窗
K查詢游標下單字的 man 頁面
ga顯示游標字元的 ASCII / Unicode 值
g8顯示游標字元的 UTF-8 Bytes
:sort排序選取行
:sort u排序並移除重複行
:sort!反向排序
:set nu顯示行號
:set rnu顯示相對行號
:set wrap啟用自動換行
:set list顯示不可見字元
:syntax on啟用語法高亮
沒有符合的指令
